# 手動測試清單 (TESTING.md)

## 1. 環境啟動與準備
- [ ] 專案根目錄下，開啟終端機執行 `pip install -r requirements.txt`。
- [ ] 執行 `uvicorn app:app --reload` 啟動 FastAPI 伺服器。
- [ ] 打開瀏覽器訪問 `http://127.0.0.1:8000`，應看到全黑質感的深色模式介面。

## 2. 帳號與 Session 測試
- [ ] 進入網頁後，系統已自動使用資料庫預設帳號 `Antigravity`，無需註冊登入。
- [ ] 點選左上角「建立新對話」，確認左邊欄會多出新的對話標籤。
- [ ] 發送該房間第一則訊息，觀察左邊欄標題是否會自動變成你所說的文字。

## 3. 檔案上傳測試 (多模態)
- [ ] 點擊對話框左側的「上傳圖示」，選擇任意圖片或文字檔。
- [ ] 發送訊息，確認畫面上出現使用者留言與附件超連結。
- [ ] 檢查專案內的 `static/uploads/` 中是否成功保存了你剛才選擇的檔案。

## 4. 生成控制 (SSE Streaming 與中止)
- [ ] 輸入長句子後送出，觀察機器人回覆是否呈現「逐字跳出」的打字機效果。
- [ ] 再輸入一句話送出，這次在生成過程中迅速點擊變為「紅色正方形」的中止按鈕。
- [ ] 確認機器人立刻停止輸出，並且後綴顯示提示「生成已由使用者中止」。

## 5. 多輪對話與歷史紀錄 (History & DB)
- [ ] 重整網頁。
- [ ] 點擊剛才說過話的對話標籤，確認先前的對話歷史 (Message Table) 都有馬上依照順序載入。
- [ ] 可以切換不同的對話標籤，確認彼此上下文內容不會互相污染。
