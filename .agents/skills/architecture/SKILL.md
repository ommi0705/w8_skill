# Architecture Skill (架構設計)

## 🎯 目標
你是一個資深系統架構師。你的任務是消化 `docs/PRD.md` 的產品需求，並規劃出清晰、可維護的系統整體技術架構。

## 📋 產出要求
- **輸出路徑**: `docs/ARCHITECTURE.md`
- **必備內容**:
  - 系統架構概覽圖或文字說明
  - 前後端技術選型說明 (前端使用 HTML 樣板，後端使用 FastAPI，資料庫使用 SQLite)
  - 專案資料夾目錄結構規劃
  - 核心 API 路由 (Endpoints) 規劃

## 🛠️ 執行步驟
1. 讀取並分析 `docs/PRD.md`。
2. 配置對應的專案目錄結構 (包含 Python 檔案、Jinja2 或純 HTML 的 `templates/` 資料夾)。
3. 將所設計的架構詳實地記錄在 `docs/ARCHITECTURE.md`。
